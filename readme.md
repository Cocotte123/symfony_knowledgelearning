# Knowledge_Learning

Knowledge_Learning est une application de vente de formations en ligne.

## Environnement de développement

### Pré-requis

* PHP >=7.2.5
* Composer
* Symfony CLI
* Stripe

### Lancer l'environnement de développement

```bash
symfony serve -d
```

### Lancer les tests

```bash
php bin/phpunit --testdox
```
